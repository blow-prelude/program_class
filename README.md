## 程序设计上机
### 题目要求：
1. 读取中文文档，其中所有标点为英文标点
2. 文档处理
   1. 英文标点转中文标点
   2. 段尾空格删除
   3. 首行缩进2个空格
3. 文档统计
   1. 段落数量
   2. 句子数
   3. 总字数
   4. 每个字出现的次数，并按照词频顺序排序
   5. 每个标点出现的次数
   6. 保存到另一个文件
4. 文档转化
   1. 将文档转化为竖排格式
   2. 每列的字数可以由用户控制
   3. 保存到文件中
